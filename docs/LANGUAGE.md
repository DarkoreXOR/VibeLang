# Vibelang Language Guide

This is the entry point for language documentation.  
The full guide is split by feature and ordered from easiest to hardest.

## How to read

- New to Vibelang: read from `01` upward.
- Returning user: jump to the feature you need.
- Each feature file includes syntax examples and common pitfalls.
